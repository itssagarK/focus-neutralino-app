# Focus Neutralino App

This is a minimalistic focus timer built using Neutralino.js.

## Features
- Simple and clean interface
- Start/stop focus timer
- Notifications

## Screenshot file:///C:/Users/Home/focus/focus-app.png.png
![Developer Focus Timer](focus-timer.png)

## Installation
1. Clone the repository.
2. Run `npm install`.
3. Start the app using `npm start`.

## Usage
- Set your timer and start focusing!
